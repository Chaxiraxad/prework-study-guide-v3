HTML

  • HTML is the standard markup language for creating Web pages.
  • HTML (HyperText Markup Language) is the code that is used to structure a web page and its content.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • CSS (Cascading Style Sheets) is used to style and layout web pages.
  • CSS is the language for describing the presentation of Web pages, including colors, layout, and fonts.

Git

  • Git is a tool that's used to manage multiple versions of source code edits that are then transferred to files in a Git repository
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript is a scripting or programming language that allows you to implement complex features on web pages.
  • Some of what JavaScript can do for the web includes: adding interactive behavior to web pages like zooming in and out or playing audio/video. creating web and mobile apps, the most popular examples consist of the likes of Netflix and Uber.
